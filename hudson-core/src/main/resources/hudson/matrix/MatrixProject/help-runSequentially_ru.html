При включении этой опции Hudson собирает каждую конфигурацию последовательно. Это может быть полезно если Ваша конфигурация нуждается в доступе к общим ресурсам (например база данных), а также для предотвращения конфликтов с другими проектами.

Для этого свойства возможно использовать Проектное каскадирование. Детали доступны в этом документе.
