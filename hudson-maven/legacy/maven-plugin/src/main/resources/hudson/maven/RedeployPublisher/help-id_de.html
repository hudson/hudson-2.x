Für manche Repository-URLs (z.B. scpexe) benötigt Maven zusätzliche Einstellungen (Benutzername, Pfad zum Kommando, usw.). Maven erwartet diese Einstellungen in der Datei ~/.m2/settings.xml des Benutzerkontos, unter dem Hudson läuft. Die hier angegebene Repository-ID wird verwendet, um die passenden Einstellungen in der Settings-Datei zu finden (technisch: die protokollspezifischen Informationen werden dem <server>-Element entnommen, dessen ID mit dem hier angegebenen Wert übereinstimmt).

Mehr dazu finden Sie unter Verwendung des maven-deploy-Plugins.
