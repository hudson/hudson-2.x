Если Ваш проект требует очистки рабочей директории перед каждой сборкой, выберите этоту опцию. В этом случае система удалит все файлы внутри рабочей директории перед началом каждой сборки. Это может быть полезно если Вы хотите получать свежую копию исходных кодов из SCM системы перед каждой сборкой.

Для этого свойства возможно использовать Проектное каскадирование. Детали доступны в этом документе.
